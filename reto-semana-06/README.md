# Reto 6: Validador de Códigos con Expresiones Regulares

## Descripción
Sistema de validación automática de códigos usando expresiones regulares para una empresa de logística.

## Tipos de códigos soportados
- **Producto:** `TEC-0001-MX`
- **Envío:** `ENV-2024-03-15-001234`
- **Empleado:** `EMP-VEN-1234`
- **Factura:** `FAC-A-123456`

## Archivos
- `reto6_validador.ipynb` — Notebook con la solución completa

## Tecnologías
- Python 3
- Módulo `re` (expresiones regulares)
